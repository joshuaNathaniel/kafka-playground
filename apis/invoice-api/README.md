# Invoice API
This project was bootstrapped with Fastify-CLI.

## Available Scripts

In the project directory, you can run:

### `yarn dev`

To start the app in dev mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `yarn start`

For production mode

### `yarn test`

Run the test cases.
